# Welcome to Artisan Sale Manager

**Artisan Sale Manager** is a professional WooCommerce plugin that revolutionizes how you manage sales campaigns. Built with a modern React interface and advanced product segmentation, it's designed for store owners who need powerful, reliable sale management tools.

## What Makes It Different

Unlike other sale plugins that only change display prices, Artisan Sale Manager updates **real WooCommerce sale prices**. This means:

- ✅ **Product feeds work correctly** (Google Shopping, Facebook Ads)
- ✅ **Third-party plugins see actual sale prices**
- ✅ **WooCommerce analytics are accurate**
- ✅ **No compatibility issues** with themes or plugins

## Key Features

### 🎯 **Campaign Management**
Create organized sale campaigns with independent rules, schedules, and product selections. Each campaign operates independently with full conflict resolution.

### 🔍 **Advanced Product Segmentation**
Target products using powerful filters:
- **Categories & Tags**: Select entire product groups
- **Stock Status**: In-stock, out-of-stock, or low-stock products
- **Product Types**: Simple, variable, grouped products
- **Custom Attributes**: Size, color, brand, or any custom field

### ⚡ **Smart Conflict Resolution**
When multiple campaigns target the same product, our priority system ensures the highest priority campaign always wins - no confusion, no overwrites.

### 📅 **Flexible Scheduling**
- **Manual Control**: Activate/deactivate campaigns instantly
- **Scheduled Campaigns**: Set start and end dates for automatic management
- **Duration Tracking**: See exactly how long campaigns have been running

### 🛡️ **Enterprise Features**
- **Real-time Updates**: Changes apply immediately across your store
- **Backup & Rollback**: Original prices are safely backed up
- **Performance Optimized**: Handles thousands of products efficiently
- **Developer Friendly**: Extensive hooks and filters for customization

## Perfect For

- **Seasonal Sales**: Black Friday, Christmas, Summer clearance
- **Flash Sales**: Limited-time promotions with precise timing
- **Category Promotions**: Target specific product categories
- **Inventory Management**: Clear old stock with targeted discounts
- **A/B Testing**: Run multiple campaigns to test different strategies

## Quick Start

1. **Install & Activate**: Get the plugin from WordPress repository
2. **Create Campaign**: Navigate to WooCommerce → Sale Manager
3. **Select Products**: Use our segment builder to target products
4. **Set Discount**: Choose percentage or fixed amount discounts
5. **Launch**: Activate immediately or schedule for later
